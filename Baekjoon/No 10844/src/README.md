# 백준 10844번 - 쉬운 계단 수

## 📝 문제 요약
- 정수 N(1 ≤ N ≤ 100)이 주어졌을 때, 길이가 N인 **계단 수의 개수**를 구하는 문제.
- **계단 수**란 인접한 모든 자리의 차이가 1인 수를 의미.
    - 예: 45656 (O), 123 (X), 210 (O)
- 단, 0으로 시작하는 수는 계단 수가 아님.
- 정답은 `1,000,000,000`으로 나눈 나머지를 출력.

---
## 예제 입력 1
```
1
```
## 예제 출력 1
```
9
```
## 예제 입력 2
```
2
```
## 예제 출력 2
```
17
```
