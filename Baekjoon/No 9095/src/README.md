# 📘 BOJ 9095: 1, 2, 3 더하기

## 🔗 링크
- https://www.acmicpc.net/problem/9095

## 📝 문제 요약
정수 `n`(1 ≤ n ≤ 11)을 1, 2, 3의 합으로 나타내는 방법의 **가짓수**를 구한다. (순서가 다르면 다른 방법)

## 📥 입력
- 첫 줄: 테스트 케이스 개수 `T`
- 다음 `T`줄: 각 테스트 케이스의 정수 `n` (1 ≤ n ≤ 11)

## 📤 출력
- 각 테스트 케이스마다 `n`을 1, 2, 3의 합으로 나타내는 방법의 수

## 입출력 예제
### 입력 예제 1
```
3
4
7
10
```
### 출력 예제 1
```
7
44
274
```
