# 🧾 백준 14501 - 퇴사

## 📌 문제 설명

- 퇴사까지 남은 N일 동안 상담 일정이 주어진다.
- 각 날짜마다 상담을 진행하는 데 걸리는 기간 `Ti`와 받을 수 있는 금액 `Pi`가 주어진다.
- 상담은 중복 없이 하루에 하나만 가능하며, 퇴사 전에만 완료돼야 한다.
- 최대 수익을 얻을 수 있도록 상담 일정을 선택하라.

---

## 💡 입력

- 첫째 줄: 퇴사까지 남은 일수 N (1 ≤ N ≤ 15)
- 둘째 줄부터 N개의 줄:
    - `Ti Pi`: i번째 날에 잡힌 상담의 기간과 금액
---
## 📤 출력

- 얻을 수 있는 **최대 수익** 출력
