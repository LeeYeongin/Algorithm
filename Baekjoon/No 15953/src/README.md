# 🏆 BOJ 15953 - 상금 헌터

## 📌 문제 설명
두 개의 페스티벌에서 등수에 따라 상금을 지급한다.  
참가자는 각각 **2017년 페스티벌 A**와 **2018년 페스티벌 B**의 순위를 하나씩 가지고 있다.  
주어진 두 등수에 따라 받을 수 있는 **총 상금(원 단위)**을 구하는 문제.

---

## 🥇 상금 규칙

### 2017년 페스티벌 (Festival A)
- 총 상금: **1,000만원**
- 등수별 상금:

| 등수 범위 | 상금(만원) |
|-----------|------------|
| 1등       | 500        |
| 2~3등     | 300        |
| 4~6등     | 200        |
| 7~10등    | 50         |
| 11~15등   | 30         |
| 16~21등   | 10         |

---

### 2018년 페스티벌 (Festival B)
- 총 상금: **1,023만원**
- 등수별 상금:

| 등수 범위 | 상금(만원) |
|-----------|------------|
| 1등       | 512        |
| 2~3등     | 256        |
| 4~7등     | 128        |
| 8~15등    | 64         |
| 16~31등   | 32         |

- **순위가 시상 범위 밖이면 0원**

---

## ⛳ 입력
1. 첫 줄에 테스트 케이스 개수 `T` (1 ≤ T ≤ 1,000)
2. 이후 각 줄에 두 정수 `a b`
    - `a`: 2017년 대회 순위 (0 ≤ a ≤ 100)
    - `b`: 2018년 대회 순위 (0 ≤ b ≤ 64)
    - (0은 참가하지 않음을 의미)

---

## 🎯 출력
- 각 테스트 케이스마다 받을 총 상금을 **원 단위**로 출력한다.  
  (상금 × 10,000)

---

## 📝 예제

### 입력
```
6
8 4
13 19
8 10
18 18
8 25
13 64
```
### 출력
```
1780000
620000
1140000
420000
820000
620000
```
