# 📘 백준 1929번: 소수 구하기

## 🔗 문제 링크
- https://www.acmicpc.net/problem/1929

## 📝 문제 요약
정수 구간 **[M, N]**(1 ≤ M ≤ N ≤ 1,000,000)에서 **모든 소수**를 출력한다. 각 소수는 한 줄에 하나씩 출력한다.

## 📥 입력
- 한 줄에 정수 `M N`

## 📤 출력
- 구간 `[M, N]`에 포함되는 **모든 소수**를 오름차순으로 한 줄에 하나씩 출력

## 🧪 입출력 예시
### 입력 예시 1
```
3 16
```
### 출력 예시 1
```
3
5
7
11
13
```
